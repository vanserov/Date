3. Служба времени. Клиент задает текстовую строку с запросом времени или даты 
(например, "дата через день", "дата через 2 недели", "время сейчас", "время через 
час") и по кнопке отправляет на сервер. Сервер анализирует запрос, рассчитывает 
время и возвращает клиенту в формате "ГГ.ММ.ДД ЧЧ:ММ:СС".
